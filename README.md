# Banking API (Golang + Gin + MySQL)

A simple Banking REST API built with **Golang**, **Gin**, and **MySQL**.  
It supports authentication (JWT), accounts, deposits, transfers, loans, loan payments, and beneficiaries.

---

## Features

- ✅ User Register + Login (JWT)
- ✅ Create & List Accounts
- ✅ Deposit Money
- ✅ Transfer Money
- ✅ Create & List Loans
- ✅ Repay Loan + List Loan Payments
- ✅ Add Beneficiary
- ✅ MySQL database integration
- ✅ Environment-based configuration using `.env`

---

## Tech Stack

- Go (Golang)
- Gin (HTTP web framework)
- MySQL
- GORM (ORM)
- JWT Authentication
